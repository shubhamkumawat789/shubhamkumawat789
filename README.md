### üèÜ Featured Project

#### ü§ü Real-Time AI Sign Language Translator
A comprehensive application designed to bridge the communication gap for the deaf and hard-of-hearing community.

- **Real-Time Detection:** Uses **MediaPipe** and **ALSTM/LSTM** models to detect and translate sign language gestures instantly via webcam.
- **Bi-Directional Communication:**
  - **Sign-to-Text/Speech:** Converts hand gestures into spoken language.
  - **Text/Speech-to-Sign:** Converts input text or speech into sign language animations.
- **Robustness:** Features a custom-trained model on 160+ signs with techniques like **OneEuroFilter** for jitter reduction and motion gating for accuracy.
- **Tech Stack:** Python, Streamlit, TensorFlow, OpenCV, MediaPipe.
